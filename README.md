# Gemini Enterprise Day 1 Value Workshop — IHG Team

Welcome to the **Gemini Enterprise Day 1 Value Workshop** tailored specifically for the **InterContinental Hotels Group (IHG)** team!

This 2-hour virtual hands-on workshop equips IHG employees across all departments—including Hotel Operations, Guest Experience, Franchise Support, Marketing, HR, IT, and Corporate Strategy—with practical skills to unlock immediate productivity using Google's **Gemini Enterprise**.

---

## 📅 Complete Workshop Agenda (13 Tasks / 2 Hours)

| Task | Module / Task Name | Key Focus & Hands-On Exercises |
| :--- | :--- | :--- |
| **Setup** | **Welcome & Prerequisites** | Enterprise Security Boundaries, Pre-Indexed Data Stores |
| **Task 1** | **Interface Navigation & Basic Features** | Omnibar, Starred Chats, Sharing Links, Translation, Code Gen |
| **Task 2** | **Personalization & Appearance** | Profile Setup (Hospitality Role), Memory & History Toggles |
| **Task 3** | **Web Search, File Analysis & Data Handling** | Google Search Grounding, PDF Upload, CSV Graph/Chart Gen, Table Export |
| **Task 4** | **Searching Internal Company Data** | Querying Enterprise Data Stores (Cymbal Bank / GCS Bucket) |
| **Task 5** | **Using @ Mentions & Direct Agent Calling** | Omnibar `@` Mentions (`@Deep Research`) |
| **Task 6** | **Strategic Ideation & Brainstorming** | Structured Ideation & Multi-Option Evaluation in Gemini Chat |
| **Task 7** | **Conducting Deep Research** | Multi-Page Cited Reports, Research Plan Review, Audio Summaries |
| **Task 8** | **Generating Media (Images & Video)** | Image Generation, Infographics from PDF, 8-sec Veo Video Gen |
| **Task 9** | **Unlocking Insights with NotebookLM** | Grounded Research, Inline Citations, Audio/Video Podcast Overviews |
| **Task 10** | **Build Agent from Prompt (Agent Designer)** | Prompt-to-Agent Builder, Daily Scheduled Briefings |
| **Task 11** | **Build Agent with Builder (Subagents)** | Multi-Step Subagents (IHG Group Sales & Event Follow-up Agent) |
| **Task 12** | **Review Submitted Tasks** | Deep Research Reports, NotebookLM Audio/Video Output Review |
| **Task 13** | **NotebookLM Challenge Labs** | Scenario Challenges (Market Analyst, GTM Synthesis, Sustainability) |
